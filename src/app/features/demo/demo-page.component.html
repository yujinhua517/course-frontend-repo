使用說明
權限控制系統：
  • 使用 PermissionGuard 保護路由，設定所需的角色或權限
  • 在組件中使用 UserStore 檢查用戶權限
  • 支援角色檢查、權限檢查、複合條件檢查
消息通知系統：
  • 使用 GlobalMessageService 顯示各類型消息
  • 支援自動消失、手動關閉、自定義操作按鈕
  • 消息顯示在螢幕右上角，支援多條消息排隊
開發建議：
  • 在生產環境中請替換模擬數據為實際 API 調用
  • 根據需求擴展角色和權限定義
  • 為所有權限邏輯編寫單元測試
